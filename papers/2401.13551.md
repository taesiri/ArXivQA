# [Interleaving One-Class and Weakly-Supervised Models with Adaptive   Thresholding for Unsupervised Video Anomaly Detection](https://arxiv.org/abs/2401.13551)

## Write a very high-quality and detailed summary of the paper that describes the paper in a way that a human can fully understand. The summary should cover the problem, the proposed solution and highlight the main contributions of the paper.

 Here is a detailed summary of the paper:

Problem:
The paper tackles the challenging problem of unsupervised video anomaly detection (UVAD). UVAD aims to identify abnormal events in videos without using any human annotated labels during training. Most prior work has focused on supervised anomaly detection using either one-class classification (OCC) or weakly-supervised (WS) approaches. However, directly applying these methods for UVAD does not work well. The only prior UVAD method (GCL) has two main limitations: (1) The OCC and WS models are closely coupled, making it hard to upgrade them. (2) It relies on fixed thresholds to generate pseudo labels which inject errors.

Proposed Solution:
The paper proposes a novel interleaved framework to combine an OCC model and a WS model with adaptive thresholding for UVAD. The key ideas are:

1. The OCC and WS models are loosely connected, allowing flexibility to replace them with other advanced models easily.  

2. A weighted OCC model is proposed to train on both normal and abnormal data, avoiding the need for thresholding.

3. An adaptive thresholding mechanism starts with a loose threshold and automatically tightens it over progressive inner-outer loops to find the optimal threshold.

Main Contributions:

1. A flexible interleaved UVAD framework that can leverage latest developments in OCC and WS anomaly detection.

2. Introduction of weighted OCC model that trains on both normal and abnormal data.

3. An adaptive thresholding method that automatically finds the optimal threshold in a progressive manner.

Experiments show the framework achieves new state-of-the-art for UVAD. When supervised labels are used, the framework achieves top results for OCC and WS anomaly detection as well, demonstrating the broad applicability.

## What is the main contribution of this paper?

 The main contributions of this paper include:

1. It proposes a novel interleaved framework for unsupervised video anomaly detection (UVAD) that alternately trains a one-class classification (OCC) model and a weakly-supervised (WS) model. The key advantage is flexibility - any OCC or WS models can be easily plugged into this framework.

2. It proposes a weighted OCC model that can be trained on both normal and abnormal data, avoiding the need to threshold the training data. 

3. It proposes an adaptive thresholding mechanism for the WS model to automatically find the optimal threshold in a progressive "loose to strict" manner. This avoids relying on unreliable manually set thresholds.

4. Experiments show the method outperforms previous UVAD methods. The OCC and WS models used in this framework are also highly competitive even when compared to supervised methods.

In summary, the main contribution is a flexible, threshold-free UVAD framework that interleaves OCC and WS models and significantly advances the state-of-the-art in unsupervised anomaly detection.


## What are the keywords or key terms associated with this paper?

 Based on my reading of the paper, some of the key terms and concepts are:

- Unsupervised video anomaly detection (UVAD) - The main task addressed in the paper, which is to detect anomalies in videos without any human annotated labels.

- One-class classification (OCC) - One of the two models used, which is trained only on normal data to identify anomalies. The paper proposes a weighted OCC model that trains on both normal and abnormal data.

- Weakly-supervised (WS) - The other model used, which leverages video-level labels to identify anomalies. The paper alternates training between OCC and WS models.  

- Adaptive thresholding - A method proposed to automatically find optimal thresholds to distinguish between normal and abnormal events, instead of using fixed manually-set thresholds.

- Interleaving framework - The overall framework that alternately trains the OCC and WS models in an inner loop, inside an outer loop that adjusts the WS threshold adaptively.

- Normal/abnormal data - The paper assumes there is more normal than abnormal data, and the relative proportions are unknown. The method tries to separate them accurately.

- Pseudo labels - Labels automatically generated by one model to train the other model, in place of human annotated labels since this is an unsupervised technique.

In summary, the key ideas involve developing an interleaving framework to train OCC and WS models, using pseudo labels and adaptive thresholding, to tackle the challenging unsupervised video anomaly detection task.
