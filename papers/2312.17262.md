# [Multimodal Classification of Teaching Activities from University Lecture   Recordings](https://arxiv.org/abs/2312.17262)

## What is the main contribution of this paper?

 This LaTeX source code does not contain a research paper. Rather, it defines a large number of journal options that can be passed to the \DeclareOption command.

Specifically, for each journal name there are commands that set the journal name (\gdef\@journal), full name (\gdef\@journalfull), short name (\gdef\@journalshort), DOI abbreviation (\gdef\@doiabbr), and ISSN number (\gdef\@ISSN). Some journals also have additional options like \gdef\@continuouspages.

So in summary, this code allows the journal metadata to be configured by passing the journal name as an option. It does not contain a research paper or contribution itself. The main contribution is defining journal metadata formatting for many MDPI journals to be used in LaTeX documents.
