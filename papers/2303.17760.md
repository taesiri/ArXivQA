## Summarize the main method used in the paper in one paragraph.

The paper proposes a novel communicative agent framework called CAMEL (Communicative Agents for "Mind" Exploration of Large Scale Language Model Society) that enables autonomous cooperation between conversational AI agents to complete tasks with minimal human involvement. The key method involves using "inception prompting" to initialize the agents and guide them through task completion while maintaining alignment with human intentions. Specifically, the framework involves a task specifier agent that takes a high-level idea from a human and makes it more concrete. The human also assigns roles like "AI assistant" and "AI user" to large language model agents. These agents are then initialized with specialized prompts that define their roles, constraints, communication protocols, and termination conditions. Once the conversation starts, the agents prompt each other automatically in a loop until the task is completed, with the AI user providing instructions and the AI assistant responding with solutions. This allows the agents to collaboratively solve tasks like developing a trading bot, while requiring only a basic idea as input from humans.By using this autonomous prompting framework, the authors generate large conversational datasets that provide insights into the behaviors and capabilities of the agent models involved. Their method offers a scalable approach to studying and improving cooperative AI systems.
