## Write a very high-quality paragraph summarizing the paper that effectively covers the paper.

 Here is a high-quality paragraph summarizing the key points of the paper:

This paper proposes a novel pose-graph optimization technique for precise vehicle localization using only a stand-alone GNSS receiver. The key idea is to use a new time-relative real-time kinematic GNSS (TR-RTK-GNSS) method to generate loop closure constraints in the pose graph, thereby reducing accumulated errors from GNSS Doppler velocity over time. Specifically, the technique leverages past GNSS measurements stored in the receiver to construct precise relative positioning baselines between non-sequential nodes. This allows closing loops in the pose graph within around 100 seconds without needing any additional sensors. Initial nodes in the graph are built up from GNSS Doppler velocities and connected via odometry-like edges. Global positioning is achieved through multi-GNSS pseudorange constraints. Experiments using a UAV with a low-cost single-frequency receiver demonstrated trajectory estimates with centimeter-level accuracy relative to ground truth, despite occasional carrier phase cycle slips. The proposed approach could enable precise, low-cost outdoor localization and mapping using only standalone GNSS receivers. Key limitations are the need for sufficient route loops within the TR-RTK-GNSS time difference limits.
