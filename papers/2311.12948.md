# [A web-based gamification of upper extremity robotic rehabilitation](https://arxiv.org/abs/2311.12948)

## Write a very high-quality paragraph summarizing the paper that effectively covers the paper.

 Here is a high-quality summary paragraph of the paper:

This paper introduces a method for gamifying upper extremity robotic rehabilitation by connecting a rehabilitation robot to web-based games. The main advantage of the proposed approach over current systems is utilizing readily available online games, reducing game development costs and providing patients and therapists more options to suit individual needs. The robotic arm has one degree of freedom and connects via a USB port to a computer, allowing control of mouse movements in games by mapping the robot's limited range of motion to mouse positions on screen. Several action web games with different difficulty levels were tested, challenging users both mentally and physically. A user study with 15 subjects playing 2 games found over 90% were satisfied that the variety of games improved engagement and motivation in rehabilitation. Results show the feasibility of gamified robotic rehabilitation using simple web browser games, providing entertainment and motivation for repetitive therapy tasks. By accessing numerous online games, more variety is enabled to decrease patient boredom.

## What is the main contribution of this paper?

 Based on my reading of the paper, the main contribution is proposing an approach for connecting a rehabilitation robot to web-based games via a USB interface. Specifically:

- They designed a user interface to easily connect the rehabilitation robot to a computer via USB. This establishes the communication between the robot and computer to control games.

- They leverage simple and freely available web browser games. By using existing web games rather than custom built games, it reduces development costs and provides a wider variety and difficulty levels of games to keep patients engaged. 

- They conducted an experiment with normal subjects using the robotic rehabilitation system connected to web games. Over 90% of subjects were satisfied that using multiple web games made the rehabilitation process more motivating and entertaining.

So in summary, the main contribution is an approach to gamify robotic upper extremity rehabilitation by connecting the robot to freely available web browser games over a USB interface. This is aimed at increasing patient motivation and engagement by providing a variety of easy to access games at low development cost.
