The key ideas are the CHORD method and COMIC dataset for category-level reconstruction.

## Summarize the paper in one paragraph.

Here is a one paragraph summary of the key points in the paper:The paper proposes a new method called CHORD (Category-level Hand-held Object Reconstruction via shape Deformation) for reconstructing the 3D shape of hand-held objects from a single RGB image. CHORD leverages a categorical shape prior and deforms it to match the observed object instance in the image. It takes a two-step approach, first predicting 2D feature maps (normals and depth) of the object using an image translation network, then deforming the 3D shape prior using an implicit function conditioned on appearance, shape, and pose-aware features. A new dataset called COMIC is introduced which contains images of categorical hand-object interactions with diverse objects, materials, poses, and viewpoints. Experiments show CHORD outperforms state-of-the-art methods on quantitative metrics and produces high-quality reconstructions, generalizing well to unseen objects. The incorporation of the three awareness types and the shape prior are shown to significantly improve reconstruction accuracy.


## Summarize the paper in two paragraphs.

Here is a two paragraph summary of the paper:The paper proposes a new method called CHORD (Category-level Hand-held Object Reconstruction via shape Deformation) for reconstructing the 3D shape of hand-held objects from a single RGB image. The key idea is to leverage a categorical shape prior to guide the reconstruction process. Specifically, CHORD first estimates the pose of a category-specific shape prior along with the hand pose. It then deforms the 2D surface feature maps (normal and depth maps) of the prior to match those of the observed object instance in the image. Using these aligned maps, it further deforms the 3D shape of the prior to reconstruct the final object surface. To enable accurate reconstruction, CHORD incorporates three types of awareness - appearance, shape, and interacting pose. Appearance awareness uses the predicted 2D maps to extract view-aligned features. Shape awareness leverages a learned structured latent code anchored on the shape prior to extract spatially-consistent features. Pose awareness represents the query point in multiple local frames of the articulated hand to capture hand-object relations. In addition, the paper introduces a new dataset called COMIC for category-level hand-object reconstruction. It contains images of humans interacting with objects from six categories rendered with diverse shapes, materials and viewpoints. Experiments show CHORD outperforms state-of-the-art methods on both synthetic and real-world images.
