# [A Roadmap Towards Automated and Regulated Robotic Systems](https://arxiv.org/abs/2403.14049)

## Write a very high-quality and detailed summary of the paper that describes the paper in a way that a human can fully understand. The summary should cover the problem, the proposed solution and highlight the main contributions of the paper.

 Here is a detailed summary of the key points from the paper:

Problem:
- Rapid development of generative AI models opens possibilities for higher automation, but uncontrolled generation poses risks, especially in safety-critical applications like medical robotics.  
- Challenge is to deploy large language models (LLMs) in robots safely and effectively given their complexity, hallucination issues, and blackbox nature.

Proposed Solution - A Roadmap:
- Decompose complex tasks into simple tasks generated by LLMs (unregulated) and regulate the workflow policy post-generation and pre-execution.
- Propose an end-to-end paradigm for fully automated and regulated robotic systems. 
- Use finite state machines (FSMs) to model state transitions and changes in "facts" in a scene. Blueprint can be inspected.
- Formalize hierarchical FSMs to represent workflows.
- Propose State Machine Serialization Language (SMSL) - a human/machine readable data format to parse FSM workflows.
- Wireless sensor network proposed for state monitoring.
- Regulation methods: Alignment of models, Inspection of plans, Supervision during execution.

Key Contributions:
- Roadmap for fully automated and regulated robotic systems
- Formalism of FSMs to model robotic tasks and workflows
- Introduction of SMSL to parse expert text to programming structures
- Approaches for state monitoring, regulation, failure handling
- Concept of human-automation continuum

The roadmap aims to achieve systems that "think like humans", "plan like humans", "move like humans" and "scrutinize like humans" through the integration of natural language models and finite state machine representations. Key goals are safe and effective automation through regulatory oversight.


## Summarize the paper in one sentence.

 This paper proposes a roadmap towards fully automated and regulated robotic systems by modeling tasks as hierarchical finite state machines, generating executable code through modular components, and integrating inspection and supervision mechanisms.


## What is the main contribution of this paper?

 According to the paper, the main contributions are:

1. A paradigmatic roadmap for an end-to-end, fully automated and regulated robotic system. The regulation includes both manual and automated methods for the automatically produced robotic system.

2. Formalism of the hFSM in modeling robotic tasks, and State Machine Seralization Language (SMSL), an open-sourced data language to parse structuralized expert text describing the controlled processes to a data structure in programming languages. 

3. A design for a Wireless Sensor Network (WSN) for state monitoring.

4. Approaches to integrate human in the loop and a concept of human-automation continuum.

In summary, the paper proposes a roadmap and methods for achieving fully automated and regulated robotic systems, including formal modeling, serialization of expert knowledge, state monitoring, and human involvement. The key innovation seems to be enabling safe and reliable automation through regulatory oversight at multiple stages.
