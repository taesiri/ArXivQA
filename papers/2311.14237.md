# [Pseudo-label Correction for Instance-dependent Noise Using   Teacher-student Framework](https://arxiv.org/abs/2311.14237)

## Write a very high-quality paragraph summarizing the paper that effectively covers the paper.

 Here is a high-quality paragraph summarizing the key points of the paper:

This paper proposes a novel teacher-student framework called Pseudo-Label Correction (P-LC) to address the challenge of learning with noisy labels. The key idea is to leverage the teacher network to correct potentially misleading noisy labels assigned to images based on pseudo-labels generated by the student network. Specifically, the teacher network is trained using triplet loss on clean data to differentiate between images of the same and different classes. The student network focuses on standard image classification. In the correction phase, the student first generates pseudo-labels for all noisy images. Then, the teacher decides whether to use the initial noisy labels or the pseudo-labels for each image based on their relative distance embeddings. Experiments on MNIST, Fashion-MNIST and SVHN show that P-LC outperforms state-of-the-art methods across all noise levels, especially for high noise, due to its self-adjustment capability and avoidance of label smoothing limitations. An additional contribution is a noise level estimator built on computing label differences between the corrected and noisy datasets. In summary, P-LC provides an effective approach for learning with noisy labels via targeted pseudo-label correction within a teacher-student framework.

2. To the best of the authors' knowledge, the first teacher-student based approach for noise level estimation.

3. Empirical experiments on MNIST, Fashion-MNIST, and SVHN demonstrating that the proposed P-LC method outperforms state-of-the-art methods across different noise levels, especially in high noise environments.

In summary, the key contribution is proposing a new teacher-student framework called P-LC that effectively handles noisy labels by correcting them using pseudo labels generated by the student network.
