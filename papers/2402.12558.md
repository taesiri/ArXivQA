- `\@journal` to "frontiersinvlsi"
- `\@journalshort` to "Frontiers in VLSI" 
- `\@journalfull` to "Frontiers in VLSI"
- `\@doiabbr` to "frontiersinvlsi"
- `\@ISSN` to the ISSN number for that journal

It covers a wide range of MDPI and Frontiers journals across many fields like computer science, engineering, medicine, chemistry, etc.

The code allows customizing the manuscript submission for many different target journals by just changing a single option. It avoids having to manually set all the metadata each time.
